# Moose <-> Wolves

This model has both the moose predicting wolves and the wolves predicting moose. 

File descriptions: 

json_two_way.ipynb is needed to create the model JSON files "two_way_nets_copy.json" and "two_way_parts_copy.json" 

two_way_sheaf.ipynb is the model with the full time series in it

Two_Way_Cells_Analysis.ipynb is the analysis of cells in the two way sheaf

sliding_two_way.ipynb is the creation and analysis of the sliding sheaf models. 

Two_way_AR_analysis.ipynb is the analysis of three different autoregressive coefficents.
